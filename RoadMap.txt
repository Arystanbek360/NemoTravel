RoadMap решил написать тут в txt формате а не писать в формате тикетов

1. Подготовка проекта
Создание репозитория на GitHub и инициализация Laravel-проекта. (2-3 минуты)
Подключение PostgreSQL к Laravel и его базовая настройка. (5-10 минут)

2. Импорт данных
Импорт JSON данных аэропортов в PostgreSQL:
Создание миграций для таблицы аэропортов. (30 минут)
Написание скрипта для загрузки данных из JSON в БД. (1-2 часа)

3. Разработка логики автокомплита (40-60 минут)
Разработка основного метода поиска аэропортов по части названия.
Оптимизация запросов для быстрой работы с большой базой данных.

4. Создание API
Реализация REST API для поиска:
Разработка маршрутов и контроллеров. (30 минут)
Написание валидации запросов и обработка ошибок. (15-30 минут)

5. Тестирование и Документирование
Написание unit и интеграционных тестов. (2 часа)
Создание Docker-файла и docker-compose. (1-2 часа)
Подготовка документации и Postman коллекции. (30 минут)
